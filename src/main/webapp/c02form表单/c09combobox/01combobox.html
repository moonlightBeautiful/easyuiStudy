下拉框组件get

input class="easyui-combobox"
    属性
  • url：String url。远程加载下拉框的数据。格式为[{"id":number,"text":String,"desc":String可选},{}...]。
  • data：array，格式为[{"id":number,"text":String,"desc":String可选},{}...]。绑定到组合框数据。
  • valueField：String。绑定到该组合框（ComboBox）的 value 上的基础数据的名称。
  • textField：String。绑定到该组合框（ComboBox）的 text 上的基础数据的名称。
  • multiple：boolean，默认值false。定义是否支持多选。
  • selectOnNavigation：boolean，默认值false。定义当通过键盘按钮上下移动选项时文本框是否显示选项。
  • panelWidth：number/'auto'，默认值null。下拉面板的宽度。
  • panelHeight：number/'auto'，默认值200。下拉面板的高度。
  • formatter：function(row每一行的数据object)，返回String，支持html标签。格式化每一行的数据。
    事件
  • onChange：function(newValue, oldValue)，当文本域的值改变的时候触发。
    方法
  • reload：String url。重新远程加载下拉框的数据。格式为[{"id":number,"text":String,"desc":String可选},{}...]。
  • disable：none。禁用组件。
  • enable：none。启用组件。
  • setValue：String。设置值valueField，如果和加载的数据匹配，则显示加载的数据项的text。如果不匹配，则添加新项value和text都是设置的值。
  • setText：String。设置值textField。
  • getValue：none。获取值valueField。
  • getText：none。获取值textField。
