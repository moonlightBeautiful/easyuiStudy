定制提示信息内容

为每个按钮定制提示信息内容.
提示信息（tooltip）组件get：
  1. 简介：当用户移动鼠标指针在某个元素/组件上时，出现提示信息窗口用来显示额外信息。
  2. 用法：div的class为easyui-tooltip。
  3. 属性：在data-options="key:value,key:value"中配置 或者 在js中$('#dd').tooltip({key:value,key:value})配置
    • data-options：↓
      • content：string，提示框内容，如果元素带有title属性，content属性没有配置的时候，提示框内容为title内容。
      • content的值也可以是func，返回 string 就可以。
      • position：string，默认值bottom，提示框位置。可能的值：'left'、'right'、'top'、'bottom'。
      • position的值也可以是func，返回 string 就可以。
