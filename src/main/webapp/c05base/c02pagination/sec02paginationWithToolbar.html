带按钮的基本分页（pagination ）组件带工具栏

定制按钮能够添加到分页工具条上去.
基本分页（pagination ）组件get：
  1. 简介：就是可拖动的元素，拖动后元素会脱离文档流，父节点的内容被抽走了，父节点大小如果没指定，会改变，先拖动的元素的z-index比后拖动的z-index大。
  2. 用法：div的class为easyui-draggable。
  3. 属性：在data-options="key:value,key:value"中配置 或者 在js中$('#dd').draggable({key:value,key:value})配置
    • data-options：↓
      • total：number，默认值1，记录总数，应该在创建分页（pagination）时设置。
      • pageSize：number，默认值1，初始页面尺寸，只能是pageList中的值。（注：每页显示的最大记录数）。
      • pageNumber：number，默认值1，创建分页（pagination）时显示的页码。
      • pageList：numberArray，默认值[10,20,30,50]，用户能改变页面尺寸。
      • showPageList：boolean，默认值true，是否显示页面尺寸列表。
      • showRefresh：boolean，默认值true，是否显示刷新按钮。
      • beforePageText：string，默认值第，在 input 组件之前显示 label。。
      • afterPageText：string，默认值共{pages}页，在 input 组件之后显示 label。。
      • displayMsg：string，默认值显示 {from}到{to},共 {total}记录，页面分页信息。
      • buttons：jsonArray,selector，定义自定义按钮z组。按钮有2个属性iconCls cssClass，它将显示一个背景图片，handler：当按钮被点击时的处理函数
