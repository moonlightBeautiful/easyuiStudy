可拖动（draggable）组件实战get：
  1. 简介：就是可拖动的元素，拖动后元素会脱离文档流。
  2. 用法：div的class为easyui-draggable。
  3. 属性：
    • revert：boolean，默认为false，如果设置为 true，拖动结束后元素将返回它的开始位置，拖动期间会脱离文档流。
    • proxy：'clone'或者none或者func，拖动时要使用的代理元素，设置为 'clone' 时，克隆元素将被用作代理。即拖动元素将由克隆产生供拖动使用，原本的元素不变。
  4. 元素的droppable方法：通过标记创建可放置（droppable）区域。
    • class="easyui-droppable"或者$('#dd').droppable({})
    • onDrop(e,source)：当可拖动元素被放下时触发。source 参数指被拖动的 DOM 元素。
    • onDragEnter(e,source)：当可拖动元素被拖进来时触发（拖进来，松开单击不算离开）。source 参数指被拖动的 DOM 元素。
    • onDragLeave(e,source)：当可拖动元素被拖离开时触发。source 参数指被拖动的 DOM 元素。
    • onDragOver(e,source)：当可拖动元素被拖过时触发，会覆盖onDragEnter和onDragLeave方法。source 参数指被拖动的 DOM 元素。
