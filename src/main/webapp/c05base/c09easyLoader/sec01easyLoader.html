加载器（EasyLoader）组件get：
  1. 简介：按需加载组件和脚本，只需要在页面中引用 jQuery 脚本 和 easyloader 脚本。
  2. 用法：easyloader.load('modules',function(){}) 或者 using('modules',function(){})。easyloader.load还有一个别名叫using，定义在window对象上。
    1、定义一个div并给它指定一个id
    2、function中使用jQuery选择器选中该div，调用模块相应的方法就可以创建EasyUI的组件
  3. 属性：
    1. easyloader.base = 'url/'。设置 easyui 基础目录，必须用'/'结束，。默认值为相对路径下的easyload.js文件。
    2. easyloader.locale = String; 预定义区域jp日本、zh_CN简体中文、zh_TW繁体中文、en英语。默认值null。
    3. easyloader.theme = String; 主题的名称预定义在themes目录下。默认值default。
    4. easyloader.css = boolean; 在加载模块的时候是否加载 CSS 文件。默认值true。
  4. 方法：
    1. load(modules,callback)：
      1. modules：模块，模块参数有效的类型包括：一个模块名称、模块名称数组、 .css"后缀结尾的CSS文件、 .js"后缀结尾的JS文件。
      2. callback：加载指定模块成功的回调函数。
