  1. 使用：可以和其他组件一起使用，为其他组件创建提示信息框。
    属性：
  1. position：string，默认值bottom。提示框位置。可能的值：'left'、'right'、'top'、'bottom'。
  2. content：string可以渲染html/func必须返回一个String/jquery对象，提示框内容。如果元素带有title属性，content属性没有配置的时候，提示框内容为title内容。
  3. trackMouse：boolean，默认值false。提示框（tooltip）是否随随着鼠标移动。 。
  4. showEvent/hideEvent：string事件，默认mouseenter和mouseleave。提示框（tooltip）出现/消失触发的事件。
  5. showDelay/hideDelay：number，默认200和100。显示/隐藏提示框（tooltip）的时间延迟。
    6. 、
    事件：
  1. onShow/onHide：e。当显示/隐藏提示框（tooltip）时触发。
  2. onUpdate：content。当提示框（tooltip）内容更新时触发，也就是提示框闪现出来时。
    方法：
  1. tip：none，返回提示对象。
  2. arrow：none，返回箭头对象。
  3. show/hide：e。显示/隐藏提示框（tooltip）。
  4. update：content。更新提示框（tooltip）内容。
