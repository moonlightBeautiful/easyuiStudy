基本窗体（window）组件get：
  1. 简介：窗口（window）是一个浮动的、可拖拽的面板，可以当做应用程序窗口使用。默认情况下，窗口可移动、可调整尺寸、可关闭。它的内容既可以通过静态 html 定义，也可以通过 ajax 动态加载。
  2. 用法：从标记创建窗口，div的class为easyui-window。使用 javascript 创建窗口，$('#div的id').window({})
  3. 属性：在标记中data-options="key:value,key:value"中配置 或者 在js中$('#div的id').window({key:value,key:value})配置
    • data-options：↓该属性扩展自面板（panel）
      • （panel）iconCls：string，标题图标。
      • （panel）tools：arrayJson,selector，自定义工具组。1、数组，每个元素包含 iconCls 和 handler 两个属性。2、选择器，指示工具组
      • title：string，标题文本。
      • collapsible：boolean ，默认值true，是否显示折叠按钮。
      • minimizable/maximizable：boolean，默认值true，是否显示最小化/最大化按钮。
      • closable：boolean，默认值true，是否显示关闭按钮。
      • closed：boolean，，默认值false，是否关闭窗口，默认窗口是打开状态。
      • draggable：boolean，默认值true，是否可拖拽。
      • resizable：boolean，默认值true，是否可调整尺寸。
      • shadow：boolean，默认值false，当窗口能够显示阴影的时候是否显示阴影。
      • modal：boolean，默认值false，是不是模态（modal）窗口，模态窗口则不允许操作源窗口。
      • inline：boolean，默认值false，定义如何放置窗口，当设置为 true 时则放在它的父容器里，当设置为 false 时则浮在所有元素的顶部。
  4. 事件：扩展自面板（panel）。
  5. 方法：扩展自面板（panel）。
    • （panel）open：none，打开面板。
    • （panel）close：none，闭面板。
