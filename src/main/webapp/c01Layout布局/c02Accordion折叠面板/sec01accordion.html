可折叠面板

可折叠面板是jquery easyui框架的一部分. 他可以让你轻松地在Web页面上定义可折叠元素.

折叠面板（Accordion）组件get：
  1. 简介：折叠面板（accordion）允许您提供多个面板（panel），同时显示一个或多个面板（panel）。
  2. 用法：从标记创建窗口，div的class为easyui-accordion。使用 javascript 创建窗口，$('#div的id').accordion({})
  3. 依赖：panel
  4. 属性：在标记中data-options="key:value,key:value"中配置 或者 在js中$('#div的id').tabs({key:value,key:value})配置
      容器属性：
      面板属性：折叠面板（Accordion）的面板（panel）属性继承自面板（panel），下面是附加的属性：
    1. selected：boolean，默认值false，是否展开面板。
    2. collapsible：boolean，默认值true，是否显示折叠按钮。
    3. 面板（panel）属性tools：array/selector，面板工具栏。
    4. 面板（panel）属性href：url，面板通过url加载内容。
  5. 方法：
    1. select：面板（panel）的标题（title）或索引（index），选择指定的面板（panel）
    2. add：面板（panel）options，添加一个新的面板（panel）。
    3. getSelected：none，获取第一个选中的面板（panel）。
    4. getPanelIndex：panelObj，获取指定的面板（panel）索引。
    5. remove：面板（panel）的标题（title）或索引（index），移除指定的面板（panel）。
