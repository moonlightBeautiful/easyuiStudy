    1. 树每个节点node数据格式：
      1. id：节点的 id，它对于加载远程数据很重要。
      2. text：要显示的节点文本。
      3. iconCls：节点图标。
      4. state：节点状态，'open' 或 'closed'，默认是 'open'。
        当设置为 'closed' 时，并未加载子节点。打开该节点若有子节点，从远程站点加载它们。
      5. checked：指示节点是否被选中。
      6. attributes：给一个节点添加的自定义属性。
      7. children：定义了一些子节点的节点数组。
      属性：
    1. url string 获取远程数据的 URL 。 null
    2. method string 检索数据的 http 方法（method）。 post
    3. animate boolean 定义当节点展开折叠时是否显示动画效果。 false
    4. checkbox boolean 定义是否在每个节点前边显示复选框。 false
    5. cascadeCheck boolean 定义是否级联选中复选框。 true
    6. onlyLeafCheck boolean 定义是否只在叶节点前显示复选框。 false
    7. lines boolean 定义是否显示树线条和展开图标下三角变成+。 false
    8. dnd boolean 定义是否启用拖放。 false
    9. data array 要加载的节点数据。[{}...]
    10. formatter function(node) 定义如何呈现节点文本。
    11. param：要传递到远程服务器的参数对象。
    12. loadFilter function(data要加载的原始数据,parentDOM 对象，表示父节点。) 返回要显示的过滤数据。返回数据时以标准树格式返回的。该函数有下列参数：
      事件：单击双击节点

    2. onBeforeLoad node, param 当加载数据的请求发出前触发，返回 false 则取消加载动作。

    15. onDrop target,source,point 当节点被放置时触发。

      方法：很多懒得看了用到再学
    1. loadData data 加载树的数据。
    2. getNode target 获取指定的节点对象。
    3. getData target 获取指定的节点数据，包括它的子节点。
    4. reload target 重新加载树的数据。
    5. getRoot none 获取根节点，返回节点对象。
    6. getRoots none 获取根节点，返回节点数组。
    7. getParent target 获取父节点，target 参数表示节点的 DOM 对象。
    8. getChildren target 获取子节点， target 参数表示节点的 DOM 对象。
    9. getChecked state 获取选中的节点。状态可用值有：'checked'、'unchecked'、。
    10. getSelected none 获取选中的节点并返回它，如果没有选中节点，则返回 null。
    11. find id 找到指定的节点并返回该节点对象。
    12. select target 选中一个节点，target 参数表示节点的 DOM 对象。
    13. check target 把指定节点设置为勾选。
    14. uncheck target 把指定节点设置为未勾选。
    15. collapse target 折叠一个节点，target 参数表示节点的 DOM 对象。
    16. expand target 展开一个节点，target 参数表示节点的 DOM 对象。当节点关闭且没有子节点时，节点的 id 值（名为 'id' 参数）将被发送至服务器以请求子节点数据。
    17. collapseAll target 折叠所有的节点。
    18. expandAll target 展开所有的节点。
    19. expandTo target 从根部展开一个指定的节点。
    20. scrollTo target 滚动到指定节点。
    21. append param 追加一些子节点到一个父节点。 parent：DOM 对象，要追加到的父节点，如果没有分配，则追加为根节点。 data：数组，节点的数据。
    22. toggle target 切换节点的展开/折叠状态，target 参数表示节点的 DOM 对象。
    23. insert param 在指定节点的前边或后边插入一个节点，param 参数包括下列属性：before：DOM 对象，前边插入的节点。 after：DOM 对象，后边插入的节点。 data：对象，节点数据。
    24. remove target 移除一个节点和它的子节点，target 参数表示节点的 DOM 对象。
    25. pop target 弹出一个节点和它的子节点，该方法和 remove 一样，但是返回了移除的节点数据。
    26. update param 更新指定的节点，'param' 参数有下列属性： target（DOM 对象，要被更新的节点）、id、text、iconCls、checked，等等。
    27. enableDnd none 启用拖放功能。
    28. disableDnd none 禁用拖放功能。
    29. beginEdit target 开始编辑节点。
    30. endEdit target 结束编辑节点。
    31. cancelEdit target 取消编辑节点。
