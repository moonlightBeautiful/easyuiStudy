  1. 列是内置的，2列， name 和 value。可进行覆盖。[[{},{}]]
    网格选项：扩展自datagrid
  1. showGroup：boolean，默认值false，是否显示属性组，属性组可以折叠或者展开同组的属性列表。
  2. groupFormatter：function(group[组的字段名],rows[属于该组的行])。格式化组字段的值。
    行数据（Row Data）：
  1. name：字段名。
  2. value：要被编辑的字段值。
  3. group：组的字段值。
  4. editor：编辑属性值的编辑器。
    事件：官网无扩展
    方法：
  1. expandGroup：groupIndex 展开指定的组。如果 'groupIndex' 参数未分配，则展开所有的组。
  2. collapseGroup：groupIndex 折叠指定的组。如果 'groupIndex' 参数未分配，则折叠所有的组。
    备注
  1. 编辑器，只有处于编辑状态的时候才能获取到。
