数据表格客户端分页

这个示例教我们如何在客户端实现数据表格分页.
其实就是把数据一次性读取出来，然后赋值给一个变量的属性，用到的时候根据页码和页面尺寸取出来赋值给rows属性

数据网格（Datagrid）组件get：
    提示：
  1. loadFilter：function，过滤器，过滤数据网格的数据。该函数有一个参数 'data'，表示原始数据。必须返回标准数据对象，含有 'total' 和 'rows' 属性。
  2. loadData：data，加载本地数据，旧的行会被移除。可以ajax获取后塞给变量，再加载。
  3. options：none，返回选项（options）对象。
  4. getPager：none，返回分页（pager）对象。
  5. pagination：boolean，默认值为false，数据网格底部是否显示分页工具栏。
    显示一个简单默认的分页工具栏，可以进行复杂配置，默认pageSize为10，分组为10,20,30,40,50，默认选择第一页。
  6. pageNumber：number，当设置了 pagination 属性为true时，初始化分页工具栏页码，会影响序号字段栏的值。默认值1。
  7. pageSize：number，当设置了 pagination 属性为true时，初始化分页工具栏页面尺寸，会影响序号字段栏的值。默认值10
  8. （分页对象事件）onSelectPage ：pageNumber, pageSize，用户选择新的页面（也就是点击按钮，pageNumber改变）或改变页面尺寸时触发。回调函数包含两个参数： pageNumber：新的页码，pageSize：新的页面尺寸
  9. （分页对象事件）refresh ：options, 刷新并显示分页信息（但是用户选择新的页面后，会自动按照目前规则刷新）。
  10. typeof 变量：String，返回变量的数据类型
    typeof "John" // 返回 string
    typeof 3.14 // 返回 number
    typeof false // 返回 boolean
    typeof [1,2,3,4] // 返回 object
    typeof {name:'John', age:34} // 返回 object
  11. Math.random()：可返回介于 0（包含） ~ 1（不包含） 之间的一个随机数，15位到17位。
  12. Math.floor(Number)：向下（左）取整计算。
  13. Math.round(Number)：向最接近取整计算。
  14. $.fn是指jquery的命名空间，加上fn上的方法及属性，会对jquery实例每一个有效。
    如扩展$.fn.abc(), 即$.fn.abc()是对 jquery 扩展了一个 abc 方法,那么后面你的每一个jquery实例都可以引用这个方法了.
    那么你可以这样子：$("#div").abc();
    $ = jQuery （可以这么理解）
  15. $.fn.datebox.defaults.formatter(new Date())：还不懂，只知道是用默认格式化日期。
  16. $.isPlainObject(data)：Boolean，否为纯粹的对象（所谓"纯粹的对象"，就是该对象是通过"{}"或"new Object"创建的。）。
  17. $.isArray(data)：Boolean，否为数组。
  18. JSON.stringify(jsonObj))：把json对象转成字符串。
  19. $.parseJSON(JSON字符串)：将符合标准格式的JSON字符串转为与之对应的JavaScript对象
  20. array对象.slice(start[,end])：从已有的数组中返回选定的元素的数组。
