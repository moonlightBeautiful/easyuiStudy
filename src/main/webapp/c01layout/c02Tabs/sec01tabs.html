      标签页/选项卡（Tabs）组件件get：
      1. 简介：每次只显示一个选项卡d的面板。
      2. 用法：从标记创建窗口，div的class为easyui-tabs。使用 javascript 创建窗口，$('#div的id').tabs({})
      3. 依赖：panel和linkbutton
      4. 属性：在标记中data-options="key:value,key:value"中配置 或者 在js中$('#div的id').tabs({key:value,key:value})配置
        • data-options：
        • （style）height：auto，则卡片高度自适应。
        • fit：boolean，默认值false，标签页容器的尺寸是否适应它的父容器。
        • plain：boolean，默认值false，是否使用背景容器图片来呈现 tab 条。
        • tools:array,selector，放置在标签栏右侧的工具栏。
        • tabPosition:string，标签页（tab）位置。可能的值：'top'、'bottom'、'left'、'right'。
        • （子div标签页面板）title：string，子div标签页面板的标题文本。
        • （子div标签页面板）iconCls:string，子div标签页面板的标题图标。
        • （子div标签页面板）closable:boolean，默认值false，是否显示一个关闭按钮。
        • （子div标签页面板）selected:boolean，默认值false，是否被选中。
        • （子div标签页面板）href:string，加载远程内容来填充标签页面板（tab panel）的 URL。
        • （子div标签页面板）tools:array,selector，标签上右侧的工具栏。
        • （子div标签页面板）content:string,selector，标签页面板的内容。
      5. 方法：
        • tabs：none，返回全部的标签页面板。
        • select：which，标签页面板（tab panel）的标题（title）或索引（index），选择一个标签页面板。
        • add：options(tab panel)，添加一个新的标签页面板（tab panel）,selected默认为true。
        • close：which，标签页面板（tab panel）的标题（title）或索引（index），关闭一个标签页面板。
        • getSelected：none，获取选中的标签页面板（tab panel）。
        • getTabIndex：tab panel，获取指定的标签页面板（tab panel）的索引。
