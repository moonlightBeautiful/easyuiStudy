调整我的尺寸



    属性：
  1. disabled：boolean，默认false。是否禁止调整尺寸。
  2. minWidth/minHeight：number，默认值10。调整尺寸时最小宽度/高度。
  3. maxWidth/maxHeight：number，默认值10。调整尺寸时最大宽度/高度。
    事件：
  • onStartResize：element。开始调整尺寸时触发。返回 false 时，DOM 元素将不进行实际的调整尺寸动作。
  • onResize：element。调整尺寸期间触发。返回 false 时，DOM 元素将不进行实际的调整尺寸动作。
  • onStopResize：element。停止调整尺寸时触发。返回 false 时，DOM 元素将不进行实际的调整尺寸动作。
    方法：
  • enable/disable：none。启用/禁用可调整尺寸。
