加载日历 加载对话框 加载数据表格



    hello：
  1. 简介：按需加载组件和脚本，只需要在页面中引用 jQuery 脚本 和 easyloader 脚本。
  2. 用法：easyloader.load还有一个别名叫using，定义在window对象上。
    easyloader.load('modules',function(){})
    using('modules',function(){})
    属性：
  1. easyloader.base：'url/'，默认值为相对路径下的easyload.js文件。easyui 基础目录，必须用'/'结束。
  2. easyloader.locale：String，默认值null。语言， 预定义区域jp日本、zh_CN简体中文、zh_TW繁体中文、en英语。
  3. easyloader.theme：String，默认值default。 加载模块的主题，名称预定义在themes目录下。
  4. easyloader.css：boolean，默认值true。 加载模块的时候是否加载 CSS 文件。
    事件：
    方法：
  • load(modules[,callback])：
    1. modules：加载的模块。模块参数有效的类型包括：一个模块名称、模块名称数组、 .css"后缀结尾的CSS文件、 .js"后缀结尾的JS文件。
    2. callback：加载指定模块成功的回调函数。
