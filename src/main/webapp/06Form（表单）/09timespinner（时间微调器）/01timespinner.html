DateTimeSpinner 日期时间微调框
值:



    选项：扩展自微调器（spinner）
  1. separator string 时分秒之间的分隔符。 :
  2. showSeconds boolean 定义是否显示秒的信息。 false
  3. highlight number 初始高亮的域，0 = 时，1 = 分，... 0
    事件：扩展自微调器（spinner）
    方法：扩展自微调器（spinner）
  1. getHours none 获取当前的时钟的值。
  2. getMinutes none 获取当前的分钟的值。
  3. getSeconds none 获取当前的秒钟的值。
