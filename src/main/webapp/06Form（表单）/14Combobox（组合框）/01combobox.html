选项：
  1. valueField：String。绑定到该组合框（ComboBox）的 value 上的基础数据的名称。
  2. textField：String。绑定到该组合框（ComboBox）的 text 上的基础数据的名称。
  3. url：String url。远程加载下拉框的数据。格式为[{"id可以取别的名":number,"text可以取别的名":String,"desc":String可选},{}...]。
  4. data：array，格式为[{"id可以取别的名":number,"text可以取别的名":String,"desc":String可选},{}...]。绑定到组合框数据。
  5. formatter：function(row每一行的数据object)，返回String，支持html标签。格式化每一行的数据。
    事件：
  1. onBeforeLoad param 在请求加载数据之前触发，返回 false 则取消加载动作。
  2. onLoadSuccess none 当远程数据加载成功时触发。
  3. onSelect record 当用户选择一个列表项时触发。。
    方法：
  1. getData：none 返回加载的数据。。
  2. loadData：data。加载本地列表数据。。
  3. reload：url。请求重新加载远程的列表数据。。
  4. setValues|setValue：values|value。设置组合框（combobox）的值。
  5. clear none 清除组合框（combobox）的值。
  6. select|unselect value|value 选择|取消选择指定的选项。
