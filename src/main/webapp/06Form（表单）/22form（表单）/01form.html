  1. 在同一个form中
    选项：
  1. url string 要提交的表单动作 URL。 null
    事件： onSubmit param 提交前触发，返回 false 来阻止提交动作。 success data 当表单提交成功时触发。 onBeforeLoad param 发出请求加载数据之前触发。返回 false 就取消这个动作。 onLoadSuccess data 当表单数据加载时触发。 onLoadError none 加载表单数据时发生某些错误的时候触发。
    方法：
  1. submit options 做提交动作,验证不通过不让提交，options 参数是一个对象，它包含下列属性： url：动作的 URL onSubmit：提交之前的回调函数，返回false，则不会提交。 success：提交成功之后的回调函数
    2. load data|url 加载记录来填充表单。data{ name:'name2', email:'mymail@gmail.com', } clear none 清除表单数据，默认值不存在。 reset none 重置表单数据，默认值存在。该方法自版本 1.3.2 起可用。 validate none 进行表单字段验证，当全部字段都有效时返回 true 。该方法和 validatebox 插件一起使用。 enableValidation none 启用验证。该方法自版本 1.3.4 起可用。 disableValidation none 禁用验证。该方法自版本 1.3.4 起可用。
