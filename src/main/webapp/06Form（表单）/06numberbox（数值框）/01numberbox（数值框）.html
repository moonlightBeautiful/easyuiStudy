选项：扩展自验证框（validatebox）
  1. buttonText：string，默认值" "。在文本框上附加的按钮显示的文本。
  2. disabled boolean 定义是否禁用该字段。 false
  3. value number 默认值。
  4. min number 允许的最小值。 null
  5. max number 允许的最大值。 null
  6. precision number 显示在小数点后面的最大精度。 0
  7. decimalSeparator string 分隔数字的整数部分和小数部分的分隔字符。 .
  8. groupSeparator string 分隔整数组合的字符。
  9. prefix string 前缀字符串。
  10. suffix string 后缀字符串。
  11. filter function(e) 定义如何过滤被按下的键，返回 true 则接受输入字符。该属性自版本 1.3.3 起可用。
  12. formatter function(value) 用来格式数字框（numberbox）值的函数。返回显示在框中的字符串值。。
  13. parser function(s) 用来解析字符串的函数。返回数字框（numberbox）值。
    事件：
  1. onChange：newValue,oldValue。当字段值改变时触发。
    方法：扩展自验证框（validatebox） disable none 禁用该域。 enable none 启用该域。 fix none 把值固定为有效的值。 setValue none 设置数字框（numberbox）的值。 代码实例： $('#nn').numberbox('setValue', 206.12); getValue none 获取数字框（numberbox）的值。 代码实例： var v = $('#nn').numberbox('getValue'); alert(v); clear none 清除数字框（numberbox）的值。 reset none 重置数字框（numberbox）的值。该方法自版本 1.3.2 起可用。
