选项：扩展自combo(自定义下拉框)和treegrid(树形表格)
  1. idField string ID字段名称。 null treeField string 要显示在文本框中的文本字段。 null textField string 要绑定到该组件对应的底层数据结构的字段名。（该属性自1.5.2版开始可用） null limitToGrid boolean 限制输入的值只能是树形表格中的值。 FALSE
    事件：完全扩展自combo(自定义下拉框)和treegrid(数据表格)。
    方法：扩展自组合（combo）
  1. grid none 返回数据表格对象。
  2. setValue value 设置组件值。 $('#cc').combogrid('setValue', '002'); $('#cc').combogrid('setValue', {id:'003',name:'name003'});
  3. setValues values 设置组件值数组。 $('#cc').combogrid('setValues', ['001','007']); $('#cc').combogrid('setValues', ['001','007',{id:'008',name:'name008'}]);
  4. clear none 清除组件的值。
