  1. 不支持标记创建，只支持js创建。
    选项：扩展自验证框（validatebox）
  1. width number 组件的宽度。 auto height number 组件的高度。该属性自版本 1.3.2 起可用。 22
  2. panelWidth number 下拉面板的宽度。 null panelHeight number 下拉面板的高度。 200
  3. multiple boolean 定义是否支持多选。 false
  4. selectOnNavigation boolean 定义当通过键盘导航项目时是否选择项目。该属性自版本 1.3.3 起可用。 true
  5. separator string 多选时文本的分隔符。 ,
  6. editable boolean 定义用户是否可以往文本域中直接输入文字。 true
  7. disabled boolean 定义是否禁用文本域。 false readonly boolean 定义组件是否只读。该属性自版本 1.3.3 起可用。 false
  8. hasDownArrow boolean 定义是否显示向下箭头的按钮。 true
  9. value string 默认值。
  10. delay number 从最后一个键的输入事件起，延迟进行搜索。 200
  11. keyHandler object 当用户按键后调用的函数。默认的 keyHandler 定义如下： keyHandler: { up: function(){}, down: function(){}, enter: function(){}, query: function(q){} }
    事件：扩展自 timespinner
  1. onShowPanel none 当下拉面板显示的时候触发。 onHidePanel none 当下拉面板隐藏的时候触发。
  2. onChange newValue, oldValue 当文本域的值改变的时候触发。
    方法：扩展自验证框（validatebox）
  1. panel none 返回下拉面板对象。 textbox none 返回文本框对象。
  2. showPanel none 显示下拉面板。 hidePanel none 隐藏下拉面板。
  3. disable none 禁用组件。 enable none 启用组件。
  4. readonly mode 启用/禁用只读模式。该方法自版本 1.3.3 起可用。 用法实例： $('#cc').combo('readonly'); // enable readonly mode $('#cc').combo('readonly', true); // enable readonly mode $('#cc').combo('readonly', false); // disable readonly mode
  5. validate none 验证输入的值。
  6. isValid none 返回验证结果。
  7. clear none 清除组件的值。 reset none 重置组件的值。该方法自版本 1.3.2 起可用。
  8. getText none 获取输入的文本。 setText none 设置文本值。
  9. getValues none 获取组件的值的数组。 setValues none 设置组件的值的数组。
  10. getValue none 获取组件的值。 setValue none 设置组件的值。
