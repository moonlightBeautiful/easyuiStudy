选项：扩展自 combobox (组合框)
  1. value：String。绑定到该标签框（ComboBox）的 value。"String,String..."
  2. hasDownArrow： boolean 定义是否显示向下的箭头。false
  3. tagFormatter ：function(value,row) 该格式化器用于格式化返回值。
  4. tagStyler：function(value,row)。该格式化器用于格式化标签样式。
    事件：
  1. onClickTag value 在点击标签框的时候触发该事件。
  2. onBeforeRemoveTag value 在移除一个标签框之前触发，返回false则取消移除操作。
  3. onRemoveTag value 在移除标签框时触发。
    方法：扩展自 combobox
