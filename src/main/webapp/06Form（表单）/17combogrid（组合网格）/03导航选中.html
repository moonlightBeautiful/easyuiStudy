选项：扩展自组合（combo）和树（tree）
  1. loadMsg string 当数据网格（datagrid）加载远程数据时显示的消息。 null
  2. idField string id 的字段名。 null
  3. textField string 显示在文本框中的 text 字段名。 null
    事件：扩展自组合（combo）和数据网格（datagrid）
    方法：扩展自组合（combo）
  1. grid none 返回数据网格（datagrid）对象。
  2. setValues values 设置组件值的数组。 代码实例： $('#cc').combogrid('setValues', ['001','007']);。
  3. setValue value 设置组件的值。代码实例： $('#cc').combogrid('setValue', '002');
  4. clear none 清除组件的值。
