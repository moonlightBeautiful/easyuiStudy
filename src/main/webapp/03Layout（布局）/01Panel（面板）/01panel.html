jQuery EasyUI框架能够让你轻松构建Web页面.

  • easyui是一套基于JQuery的用户界面插件集合.
  • easyui为构建现代流行的交互式体验JavaScript应用程序提供了基本功能.
  • 使用easyui你不需要写很多javascript代码，你通常只需要写一些html标签来定义用户界面.
  • 完美支持HTML5.
  • easyui能够有效地节省你的开发时间.
  • easyui很简单但是很强大.

    hello：
  1. 简介：由 头部标题栏、工具栏 和 主体内容栏 组成。
    属性：
  1. title:string，默认值null。在面板头部显示的标题文本。
  2. iconCls:string，默认值null。在面板头部显示的标题文本左边显示的16x16的图标。
  3. width/height:number，默认值auto。面板宽度/高度。
  4. fit:boolean，默认值false。面板的尺寸调整到父容器大小来填充父容器，父容器有其他元素会影响。
  5. border：boolean，默认值true。是否显示面板的边框。
  6. noheader：boolean，默认值false。是否不创建面板的头部。
  7. collapsible/closable/minimizable/maximizable：boolean，默认值false。是否显示可折叠按钮/关闭按钮/最小化按钮/最大化按钮。
  8. collapsed/closed/minimized/maximized：boolean，默认值false。定义初始化面板是否折叠/关闭/最小化/最大化。
  9. loadingMessage：string。当加载远程数据时在面板主体里显示一条信息。
  10. tools：array/selector，默认值是null。自定义工具栏，
    数组，每个数组元素包含text(非必须)、iconCls、handler属性。
        tools:[ { iconCls:'icon-add', handler:function(){alert('add')} },{ iconCls:'icon-edit', handler:function(){alert('edit')} }]
    选择器，指示工具组。
        tools:'#tt'。 <div id="tt"> <a href="#" class="icon-add" onclick="javascript:alert('add')"></a> <a href="#" class="icon-edit" onclick="javascript:alert('edit')"></a> </div>
  11. href：URL string，默认值null。。面板打开后加载远程数据并且显示在面板主体里。这对创建一个惰性加载的面板（panel）很有用
    事件：
  • onBeforeOpen：none。面板打开前触发，返回 false 就停止打开。
  • onOpen：none。面板（panel）打开后触发。
  • onBeforeClose：none。面板（panel）关闭前触发，返回 false 就取消关闭。
  • onClose：none。面板（panel）关闭后触发。
  • 还有其他的各种事件：销毁前触发、销毁后触发、折叠前触发、折叠后触发、展开前触发、展开后触发、窗口最大化后触发、窗口最小化后触发、窗口还原为它的原始尺寸后触发、移动后触发、调整尺寸后触发等等
    方法：
  • options：none。返回面板的选项。
  • open：true/none。打开面板，当参数设置为 true 时，就绕过 onBeforeOpen 回调函数打开面板（panel）。 原理：display：block/none，在页面还是存在的，占用内存。
  • close：true/none。关闭面板，当参数设置为 true 时，就绕过 onBeforeClose 回调函数打开面板（panel）。 原理：display：block/none，在页面还是存在的，占用内存。
  • destroy：true/none。销毁面板。当参数设置为 true 时，就绕过 onBeforeDestroy 回调函数销毁面板（panel）。 原理：在页面中面板节点被销毁，查找不到了，不占用内存。
  • refresh：href。刷新面板（panel）加载远程数据，默认解析html标签。
  • 还有其他的各种方法：折叠面板主体、展开面板主体、把面板还原为它原来的尺寸和位置、最小化面板、最大化面板适应它的容器的尺寸、移动面板到新位置、设置面板尺寸和位置、设置头部的标题文本、获取主体对象、获取头部对象、获取面板对象、获取面板选项对象
