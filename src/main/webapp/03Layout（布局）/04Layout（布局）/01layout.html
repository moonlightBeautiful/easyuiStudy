    容器选项：
  1. fit：boolean，默认为false。布局的尺寸是否适应它的父容器。当在 'body' 标签上创建布局（layout）时，它将自动最大化到整个页面的全部尺寸。
    面板选项：
  1. title：string。显示在面板（panel）头部的标题文字。
  2. border：boolean，默认值true。是否显示面板（panel）的边框。
  3. split：boolean，默认值false。是否显示拆分栏，可以改变面板（panel）的尺寸。
  4. iconCls：string。面板（panel）头部显示一个图标的 CSS class。
  5. region：string。定义布局面板（layout panel）的位置，其值是下列之一：north、south、east、west、center。
  6. collapsible：boolean，默认值true。是否显示折叠按钮。

    方法：
  1. add：options。添加一个指定的区域面板（panel）。
  2. remove：String，值为区域north、south、east、west。移除一个指定的区域面板（panel），不能移除中center。
