jQuery EasyUI框架能够让你轻松构建Web页面.

  • easyui是一套基于JQuery的用户界面插件集合.
  • easyui为构建现代流行的交互式体验JavaScript应用程序提供了基本功能.
  • 使用easyui你不需要写很多javascript代码，你通常只需要写一些html标签来定义用户界面.
  • 完美支持HTML5.
  • easyui能够有效地节省你的开发时间.
  • easyui很简单但是很强大.

    容器选项：
  1. fit：boolean，默认值false。标签页容器的尺寸是否适应它的父容器。
  2. plain：boolean，默认值false。是否使简化 tab 条。
  3. tools:array,selector。放置在标签栏右侧的工具栏。
  4. toolPosition:'left'、'right'。工具栏位置。
  5. selected：面板index，0开始。初始化选中的面板。
  6. tabPosition:string。标签页（tab）位置。可能的值：'top'、'bottom'、'left'、'right'。
    面板选项：继承panel
  1. title：string。标签页面板（tab panel）的标题文字。
  2. content：string。签页面板（tab panel）的内容。
  3. href：string。加载远程内容来填充标签页面板（tab panel）的 URL。。
  4. iconCls：string，默认值null。显示在标签页面板（tab panel）标题上的图标的 CSS class。
    事件：
  1. onSelect(title/index)：当用户选择一个标签页面板（tab panel）时触发。
    方法：
  1. tabs：none。返回全部的标签面板的面板。
  2. exists：面板的标题（title）或索引（index）。是否存在，返回boolean。
  3. select：面板的标题（title）或索引（index）。选择指定的面板打开。
  4. add：面板选项。添加一个新的标签页面板,selected默认为true。
  5. close：标签页面板的面板的标题（title）或面板的索引（index）。关闭指定的标签页的面板。
  6. getSelected：none。获取选中的标签页面板（tab panel）。
  7. getTabIndex：tab panel。获取指定的标签页面板（tab panel）的索引。
