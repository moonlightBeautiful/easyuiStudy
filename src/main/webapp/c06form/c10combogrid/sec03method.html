下拉表格组件get

input class="easyui-combogrid"
简介：
  • 表单组件的一种
  • 下拉表格组件（combogrid）把可编辑的文本框和下拉数据网格面板结合起来，用户可以从下拉数据网格面板中快速查找和选择。下拉表格组件（combogrid）提供了选择某个项目的键盘导航支持。
  • 下拉表格组件（combobox）可以是用select标签（远程加载数据）或者 input标签（远程加载数据）创建。
  • 属性：
    • data-options↓
      • panelWidth：number。下拉表格面板的宽度。
      • panelHeight：number。下拉表格面板的高度。
      • idField：string。id 的字段名。。
      • textField：string。显示在文本框中的 text 字段名。
      • url：string。从远程加载列表数据的 URL ，返回的数据是json对象，有rows（json数组）和total属性（number）。
      • columns：jsonArray [[json对象1，json对象2，json对象3..]]，field：url返回的json对象的属性id， title：下拉表格的标题，width：下拉表格字段宽度。下拉表格的字段配置。
      • fitColumns：boolean，默认false。是否自适应宽度。
      • multiple：boolean，默认false。定义是否支持多选。
      • selectOnNavigation：boolean，默认true。定义当通过键盘上下键导航项目时是编辑框否选择项目。
    • value：string，远程加载数据的id字段的值。文本框的默认值。
  • 方法：
    • getValue：none。获取编辑框的值的id字段。
    • setValue：string，对应url返回数据的id。设置编辑框的值。
    • getValues：none。获取编辑框的值的id字段s。
    • setValues：stringArray，对应url返回数据的ids。设置编辑框的值s，对应设置多选的情况。
    • disable：none。文本框不可用。
    • enable：none。文本框可用。
