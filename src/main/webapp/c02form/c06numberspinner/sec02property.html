数字微调器组件

increment：点击微调按钮按照指定量数调整时间.

数字微调器组件get

input class="easyui-numberbox"
简介：
  • 表单组件的一种
  • 数值微调器（numberspinner）是基于微调器（spinner）和数字框（numberbox）创建的。它允许用户使用向上/向下微调按钮滚动到一个期望值。
  • 属性：扩展自easyui-numberbox，所一easyui-numberbox有的属性它都有
    • data-options：可以在标签中直接使用data-options赋值，也可以通过选择器设置、改变$('#日历id').calendar({属性:值})
      • increment：number。调整数字的增量.
