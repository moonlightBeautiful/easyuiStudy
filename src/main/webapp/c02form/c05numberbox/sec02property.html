数字框组件get

input class="easyui-numberbox"
简介：
  • 表单组件的一种
  • 数字框（numberbox）用于让用户仅能输入数字的值，输入非数字，离开输入框清空输入的内容。
  • 属性：
    • data-options：可以在标签中直接使用data-options赋值，也可以通过选择器设置、改变$('#日历id').calendar({属性:值})
      • required：boolean。可以在标签内，不赋值，默认true。可以放到data-options中
      • min：number。允许的最小值。
      • max：number。允许的最大值。
      • precision：number。显示在小数点后面的最大精度。
      • decimalSeparator：string。分隔数字的整数部分和小数部分的分隔字符。
      • groupSeparator：string。分隔整数组合的字符，三个一组。
      • prefix：string。前缀字符串。。
      • suffix：string。后缀字符串。。
