下拉框组件get

input class="easyui-combobox"
简介：
  • 表单组件的一种
  • 下拉框（combobox）显示一个可编辑的文本框和下拉列表，用户可以从下拉列表中选择一个或多个值,用户也可以在文本框输入，会自动匹配下拉列表中的值。
  • 下拉框（combobox）可以是用select标签（带预定义值）和input标签（远程加载值）创建。
  • 属性：
    • data-options↓
      • multiple：boolean，默认false。定义是否支持多选。
      • panelWidth：number 或者 auto。下拉面板的宽度。
      • panelHeight：number 或者 auto。下拉面板的高度。
      • selectOnNavigation：boolean，默认true。定义当通过键盘上下键导航项目时是编辑框否选择项目。
      • url：string。从远程加载列表数据的 URL ，返回的数据是json数组。
      • valueField：string。绑定远程加载的数据到该组合框（ComboBox）的 value 上的基础数据的名称。
      • textField：string。绑定远程加载的数据到该组合框（ComboBox）的 text 上的基础数据的名称。
      • formatter：func(item)，预定义数据item的属性text是文本值，value是标签的value属性值，如果是远程加载数据则item .json对象属性使用。定义如何呈现每一项。
  • 方法：
    • reload：url。input创建的下拉框专用，请求远程的列表数据，传 'url' 参数来重写原始的 URL 值。
