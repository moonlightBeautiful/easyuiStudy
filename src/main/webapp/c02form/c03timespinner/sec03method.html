时间微调器组件

时间微调器的相关操作.
获取值 设置值 禁用 启用

时间微调器组件get：

input class="easyui-timespinner"
简介：
  • 表单组件的一种
  • 允许用户通过点击组件右侧的小微调按钮来增加或减少时间
  • 属性：
    • data-options：
      • min：'08:30'。时间下限。
      • max：'18:00'。时间上限。
  • 方法：$('#时间微调器id').timespinner('方法名');
    • getValue：09:45。获取值。
    • setValue：09:45。设置值。
    • disable：。文本框不能操作。
    • enable：。文本框能操作。
