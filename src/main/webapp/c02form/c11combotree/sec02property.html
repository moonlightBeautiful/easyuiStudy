下拉树组件

input创建下拉树组件.

下拉树组件get

input class="easyui-combotree"
简介：
  • 表单组件的一种
  • 下拉树组件（combotree）把选择控件和下拉树结合起来。它与组合框（combobox）相似，不同的是把列表替换成树组件。组合树（combotree）支持带有用于多选的树状态复选框的树。
  • 下拉树组件（combotree）可以使用select标签（远程加载数据）或者 input标签（远程加载数据）创建。
  • 属性：
    • value：string，远程加载数据的id字段的值。文本框的默认值。
    • data-options↓
      • multiple：boolean，默认false。定义是否支持多选。也可以直接作为标签属性。
      • required：boolean。定义是否字段必须被输入，默认提示是该输入项为必输项。也可以直接作为标签属性，没有值。
      • url：string。从远程加载列表数据的 URL ，返回的数据是json数组，有id，text，children，state，checked，attributes等属性
      • cascadeCheck：boolean，默认true。multiple开启的时候才可以使用。也可以直接作为标签属性，值为"true"或者"false"。
