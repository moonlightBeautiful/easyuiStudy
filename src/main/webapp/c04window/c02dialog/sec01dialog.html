基本对话框（dialog）组件get：
  1. 简介：对话框（dialog）是一个特殊类型的窗口，它在顶部有一个工具栏，在底部有一个按钮栏。
  2. 用法：从标记创建窗口，div的class为easyui-dialog。使用 javascript 创建窗口，$('#div的id').dialog({})
  3. 属性：在标记中data-options="key:value,key:value"中配置 或者 在js中$('#div的id').dialog({key:value,key:value})配置
    • data-options：↓该属性扩展自窗口（window）
      • （panel）iconCls：string，标题图标。
      • （panel）tools：arrayJson,selector，自定义工具组。1、数组，每个元素包含 iconCls 和 handler 两个属性。2、选择器，指示工具组
      • （window）title：string，标题文本。
      • （window）collapsible：boolean ，默认值true，是否显示折叠按钮。
      • （window）minimizable/maximizable：boolean，默认值true，是否显示最小化/最大化按钮。
      • （window）closable：boolean，默认值true，是否显示关闭按钮。
      • （window）closed：boolean，，默认值false，是否关闭窗口，默认窗口是打开状态。
      • （window）draggable：boolean，默认值true，是否可拖拽。
      • （window）resizable：boolean，默认值true，是否可调整尺寸。
      • （window）shadow：boolean，默认值false，当窗口能够显示阴影的时候是否显示阴影。
      • （window）modal：boolean，默认值false，是不是模态（modal）窗口，模态窗口则不允许操作源窗口。
      • （window）inline：boolean，默认值false，定义如何放置窗口，当设置为 true 时则放在它的父容器里，当设置为 false 时则浮在所有元素的顶部。
      • toolbar：arrayJson/selector，对话框的顶部工具栏。按钮有三个属性text、iconCls、handler。
      • buttons：arrayJson/selector，对话框的底部工具栏。按钮有三个属性text、iconCls、handler。
  4. 事件：扩展自窗口（window）。
  5. 方法：扩展自窗口（window）。
    • （panel）open：none，打开面板。
    • （panel）close：none，闭面板。
